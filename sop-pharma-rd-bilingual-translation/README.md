# SOP医药研发中英翻译 Skill

## Skill目的

本 skill 用于将“已定稿的中文 Word SOP”转换为“中英双语 Word SOP”，用于医药研发/临床试验场景。

核心目标：
- 不改动原文，仅在复制件上翻译
- 按章节与子章节分步执行
- 按固定版式输出双语
- 最终通过格式与质量校验

## 适用场景

- 输入文件为 `.docx`
- 输入内容为已定稿中文 SOP
- 需要正文从头到尾转为中英双语
- 常见章节顺序：目的、范围、职责、流程

## 操作流程

1. 在目标文件夹定位待翻译 SOP 文档。
2. 复制原文文档并重命名为 `原名翻译版.docx`。
3. 通读正文，识别章节/子章节结构。
4. 按章节、子章节顺序翻译（一般到 `2.1/3.1/3.2` 级）。
5. 按标题规则处理所有标题。
6. 按段落规则处理非表格正文。
7. 按表格规则处理标题行与非标题行单元格。
8. 统一应用英文排版要求。
9. 运行 QA 清单并输出结果。

## 关键翻译与排版规则

标题：
- 格式：`标题号 英文 原中文`
- 英文与中文同一行
- 所有标题最终加粗

段落正文：
- 逐段翻译
- 每个原段落输出两段：英文在前、中文在后
- 英文与中文之间必须分段

表格正文：
- 标题行单元格：`英文 中文`（单行）
- 非标题行单元格：英文在前、中文在后
- 若单元格内有多段，输出为“全部英文段落在前 + 全部中文段落在后”

术语与准确性：
- 使用专业医学/临床试验术语
- 保持原文逻辑和结构
- 保证前后一致性与准确性

## 技术环境要求

- 操作系统：Windows
- 优先工具：Python + `python-docx`
- 英文样式：Times New Roman、10.5 pt（五号）、黑色
